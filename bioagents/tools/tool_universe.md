# Available Tools [¶](https://zitniklab.hms.harvard.edu/ToolUniverse/tools/tools_config_index.html\#available-tools "Link to this heading")

ToolUniverse provides over 600 scientific tools organized into local and remote categories. This overview helps you discover and navigate the available tools for your research needs.

## Tool Registry [¶](https://zitniklab.hms.harvard.edu/ToolUniverse/tools/tools_config_index.html\#tool-registry "Link to this heading")

### Local Tools [¶](https://zitniklab.hms.harvard.edu/ToolUniverse/tools/tools_config_index.html\#local-tools "Link to this heading")

Local tools are Python classes that run within the ToolUniverse process, providing high performance and seamless integration.

**Key Features:**
\- High performance with no network overhead
\- Automatic discovery and registration
\- Full access to ToolUniverse features
\- Easy development and testing

### Remote Tools [¶](https://zitniklab.hms.harvard.edu/ToolUniverse/tools/tools_config_index.html\#remote-tools "Link to this heading")

Remote tools integrate external services, APIs, and specialized systems running on different servers.

**Key Features:**
\- Integration with external APIs and services
\- Scalable computation offloading
\- Support for different programming languages
\- Secure isolation of sensitive operations

**Discover Available Tools:**

- [Tool Listing Tutorial](https://zitniklab.hms.harvard.edu/ToolUniverse/guide/listing_tools.html) \- Browse tools by category and explore the full catalog
