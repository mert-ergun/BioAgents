BioAgents

A Modular, Hierarchical Multi-Agent System for Automated Computational Biology Workflows

school Hacettepe University science Biological Data Science Lab
warning

The Problem

Exponential biomedical data growth has shifted the research bottleneck from generation to interpretation and computation. Fragmented, siloed tools and generic LLM agents struggle to reliably execute multi-step, high-stakes scientific tasks.

lightbulb

Our Solution

BioAgents is a modular, hierarchical Multi-Agent System designed to automate complex computational biology workflows, providing seamless access to integrated scientific tools and databases to significantly accelerate research.

System Architecture

Hierarchical MAS built on LangGraph with Supervisor-Specialist orchestration

Fig 1: Comprehensive System Architecture — Supervisor-Specialist orchestration, ACE self-evolution engine, and the Integrated Tool Universe.

Dynamic Tool Synthesis

The Tool Builder Agent autonomously generates and registers Python tools to bridge functional gaps in the Tool Universe.

BioAgents targets an 80% reduction in analysis time with ≥90% accuracy through automated, citation-linked workflows.

Built with FastAPI and Tailwind CSS with 3D molecular visualization via NGL.js

Workflow Builder Tab — Canvas-based drag-and-drop interface with 140+ bioinformatics node types for custom pipelines.

medication Case Study

Automated Drug Discovery Workflow

Evaluated on designing a de novo binder for the human tumor suppressor protein p53 (P04637) using an automated GPU-intensive workflow.

1

Target Characterization

Fetching UniProt records, assessing structural readiness via AlphaFold integration and biochemical profiling.

2

Hierarchical Screening

Parallel pipeline for sequence retrieval, virtual docking, and binding affinity predictions.

3

ADMET & Safety

Rigorous ADMET profiling and off-target analysis for metabolic stability and minimized toxicity.

4

Decision Logging

Transparent logs with rationalized verdicts based on gate thresholds and 3D molecular visualizations.

The pipeline combines RFdiffusion for backbone generation, ProteinMPNN for sequence design, and AlphaFold-Multimer for structural validation and binding prediction. The Protein Design Agent automatically analyzes interface quality metrics and generates reproducible technical reports with molecular visualizations and reasoning traces.
